Detailed Description

Factory class for creating MuxerListeners. Will produce a single muxer listener that will wrap the various muxer listeners that the factory supports. Currently the factory supports:

  • Media Info Dump
  • HLS
  • MPD

The listeners that will be combined will be based on the parameters given when constructing the factory.
